Belajar Git

Konfigurasi Awal

  • Setting warna : git config --global color.ui auto
  • Setting username : git config --global user.name "Lukman Hakim"
  • Setting email : git config --global user.email lukman_010395@yahoo.com

Penggunaan sehari-hari

  • Mendaftarkan file yang akan disimpan : git add namafile
  • Menyimpan file ke repository : git commit
  • Melihat history singkat semua branch dengan gambar : git log --oneline --all --graph
  • Kembali ke versi sebelumnya : git checkout commit-id

Branch dan Merge

  • Membuat branch baru : git branch nama branch
  • Pindah Branch : git checkout namaBranch
  • Menggabungkan Branch : git merge namaBranch
